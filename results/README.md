# Results

- `raw_outputs/` — every model response, one `.md` per generation, named
  `{model}__{prompt}__{task_id}.md` with a YAML front-matter metadata block.
  Produced by `harness/run_experiment.py`.
- `scores/` — one filled rubric per output, `{model}__{prompt}__{task_id}.yaml`,
  following the template in `rubric/AB_TESTING_RUBRIC.md`. A copy of that template
  lives here as `_TEMPLATE.yaml.example`.
- `REPORT.md` — generated by `harness/score_report.py`: the 2×2 totals, within-model
  before/after deltas, and the per-criterion breakdown.

Workflow: **run → score → report.** Only `*.yaml` files (not `.yaml.example`) in
`scores/` are aggregated.
